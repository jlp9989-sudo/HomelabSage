{# Cron expression widget — quick-pick select + raw input. Why both: most users want "every 6 hours", "daily at 9", or "weekly", which are awkward to type as `0 */6 * * *`. The select sets the value of the text input on change; the text input is the only field actually submitted, so a user editing the raw expression keeps full power. No HTMX, no JS framework — one inline event handler and we're done. #}
Format: min hour day month weekday. crontab.guru explains and previews any expression.
