{# Timezone widget — quick-pick select + raw text input. IANA timezone names are not exactly memorable for non-technical users. We surface the ~15 most common homelab timezones in a select; picking one sets the text input, the text input is what gets submitted. Custom names (e.g. `Africa/Cairo`, `Pacific/Honolulu`) work via the text input — full IANA database is too long to list inline and server-side validation rejects typos anyway. #}
